## Key Implementation Facts

- `keywordLW`/`keywordLC` filter by `well.get_lgr_well_tag()` — wells must be declared via `WELSPECL` not `WELSPECS`
- `COMPDATL` not needed for node generation
- `"size": 1` in JSON schema = single record; no `"size"` = table format
- `PaddedOutputString<8>` has no `operator==` or `operator<` — use `std::string(s.c_str())`
