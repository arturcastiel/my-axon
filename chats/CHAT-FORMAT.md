# CHAT FORMAT
> Template for individual chat files. Every chat file follows this structure.
> Do not edit this file — copy it via new-chat.

```markdown
# CHAT: [name]
Folder:      [folder]
Created:     [YYYY-MM-DD]
Last-active: [YYYY-MM-DD HH:MM]
Status:      active | paused | archived
Goal:        [one sentence — what this chat is for]

## CONTEXT
[Key W: values to restore when switching to this chat.
 Written automatically by chat-switch when leaving this chat.]

key: value
key: value

## PINNED
[Facts or decisions that should always be visible in this chat.
 Written by user or by register-preference scoped to this chat.]

## HISTORY
[Last 10 significant exchanges, outputs, or decisions — most recent first.
 Appended automatically when chat is paused or switched away from.]

- [YYYY-MM-DD HH:MM] [summary of what happened]
```
